Available commands for GServer:
/reloadweapons: Reloads the file weapons.txt
/shutdown: Shuts down the server
/cachesubdirs: Caches sub dirs in world (use after adding a new dir)
/reloadservermessage: Reloads the file servermessage.html
/banip IPRANGE: IP bans the IPRANGE from the server
/setjail LEVELNAME: Adds the LEVELNAME to the jail list
/addguild NAME: Adds a new guild
/removeguild NAME: Removes the guild
/addguildmember NAME ACCOUNT: Adds the account to the guild
/removeguildmember NAME ACCOUNT: Remove the account from the guild
